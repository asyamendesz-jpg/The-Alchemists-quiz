# Квиз «Какой вы логист Шёлкового пути?»

Интерактивный квиз из 5 вопросов о стиле мышления в логистике. По итогам пользователь получает один из трёх типов:

- **Мастер караванной логистики** — стратегический, расчётливый подход
- **Авантюрист Великого пути** — готовность к риску ради большой выгоды
- **Осторожный хранитель каравана** — безопасность и устойчивость

Проект не требует сборки и работает как статический HTML/CSS/JS.

## Быстрый старт

### Вставка в WordPress

1. Откройте файл [`квиз.html`](квиз.html).
2. Скопируйте **весь** его содержимое целиком.
3. В редакторе WordPress добавьте блок **«Произвольный HTML»** (Custom HTML).
4. Вставьте скопированный код и сохраните страницу.

Квиз изолирован префиксом классов `sp-quiz__*`, поэтому не должен конфликтовать со стилями темы.

## Как работает логика

Каждый ответ относится к одному из трёх типов: `risk`, `smart` или `safe`. После 5 вопросов подсчитываются баллы, и показывается результат с наибольшим количеством ответов. При равенстве приоритет такой:

1. `smart` (мастер логистики)
2. `risk` (авантюрист)
3. `safe` (хранитель каравана)

## Редактирование

| Что изменить | Где |
|---|---|
| Тексты вопросов и ответов | Массив `questions` в `квиз.html` |
| Результаты квиза | Функция `showResult()` |
| Цвета и оформление | CSS-переменные в блоке `<style>` (секция `.sp-quiz`) |

## Технические детали

- Чистый JavaScript без зависимостей
- JS обёрнут в IIFE — не засоряет глобальную область WordPress
- Адаптивная вёрстка (breakpoint 768px)
- Рассчитан на один экземпляр квиза на странице
